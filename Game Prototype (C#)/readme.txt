Comandos:

A, S, D, W -> andar
Mouse 2 -> apontar
Mouse 1 -> disparar (ao usar o mouse 2, tem maior precisão)
E -> Interagir
Esc -> Opções

Menu inicial:
-> Botão para iniciar o jogo
-> Opções
-> Sair do jogo
-> Melhores tempos

1º desafio -> acertar no alvo com a arma
2º desafio -> passar pelas madeiras até a próxima ilha
3º desafio -> fugir do tigre e pegar a chave que ele protege (atenção: o tigre não ataca, ele empurra a pessoa para cair na água e perder)
4º desafio -> encontrar o tesouro no labirinto (cuidado as paredes dão bounce)
